---
title: "Account Labels API"
api: "GET /api/base/label/{address}"
description: "Retrieve user-friendly labels for Base EOAs (Externally Owned Accounts) and Smart Contracts. Labels can be incredibly useful for categorizing your Base interactions."
---

### Inputs

<ParamField path="address" type="string" required>
  Specify up the Base addresse for which you want to retrieve labels.
</ParamField>
<ParamField query="apikey" type="string" required>
  Your API key for authentication.
</ParamField>
<ParamField query="limit" type="string">
  Specify number of shown results.
</ParamField>
<ParamField query="offset" type="string">
  Specify the starting point in the list of results for pagination. e.g.
  `offset=0`, `offset=1`, etc..
</ParamField>

### Response

<ResponseField name="data" type="array">
  An array that encapsulates the labels corresponding to the base address provided.

<Expandable title="Label Object">

<ResponseField name="blockchain" type="string">
  Name of the blockchain network.
</ResponseField>

{" "}

<ResponseField name="address_name" type="string">
  The designated name associated with the specific base address.
</ResponseField>

{" "}

<ResponseField name="address" type="string">
  The Base address for the label.
</ResponseField>

{" "}

<ResponseField name="label_type" type="string">
  Categories like 'token', 'exchange', etc., to define the nature of the label.
</ResponseField>

{" "}

<ResponseField name="label_subtype" type="string">
  Additional attributes, like token_contract_address or exchange_name, for a
  more granular understanding of the label.
</ResponseField>

{" "}

<ResponseField name="label" type="string">
  The general name or title of the label.
</ResponseField>

  </Expandable>
</ResponseField>
