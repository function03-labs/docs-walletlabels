---
title: "Search By Label"
api: "GET https://api-c.walletlabels.xyz/ethereum/search"
description: " This endpoint enables you to search and retrieve labels by their names. It's particularly useful for finding specific Ethereum accounts based on labels."
---

### Authorization

<ResponseField
  name="x-api-key"
  type="string"
  default="demo"
  initialValue="demo"
  required
>
  Your API key for authentication.
</ResponseField>

### Query Params

<ParamField query="searchtext" type="string" required>
  Specify the name or part of the name of the label you wish to search for.
</ParamField>

### Response

<ResponseField name="data" type="array">
  An array that encapsulates the labels corresponding to the Ethereum address provided.

<Expandable title="Label Object">

<ResponseField name="address_name" type="string">
  The designated name associated with the specific Ethereum address.
</ResponseField>

{" "}

<ResponseField name="address" type="string">
  The Ethereum address for the label.
</ResponseField>

{" "}

<ResponseField name="label_type" type="string">
  Categories like 'token', 'exchange', etc., to define the nature of the label.
</ResponseField>

{" "}

<ResponseField name="label_subtype" type="string">
  Additional attributes, like token_contract_address or exchange_name, for a
  more granular understanding of the label.
</ResponseField>

{" "}

<ResponseField name="label" type="string">
  The general name or title of the label.
</ResponseField>

  </Expandable>
</ResponseField>

<RequestExample>

```bash
curl --location --request GET 'https://api-c.walletlabels.xyz/ethereum/search?searchtext=Uniswap' \
--header 'Content-Type: application/json' \
--header 'x-api-key: demo'

```

</RequestExample>
